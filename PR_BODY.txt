## Summary

Implements flexible CLI argument handling with named options alongside existing positional arguments, and bumps version to 1.2.0 per approved plan.

## Changes

**CLI Enhancements** (`src/diff_utility/cli.py`):
- Help options: `/?` and `--?` now work alongside `--help`
- Named file inputs: `-file1`/`-1` for first file, `-file2`/`-2` for second file
- Output to file: `-output`/`-o` writes results to specified file instead of stdout
- Named arguments override positional when both provided (using explicit `is not None` checks)
- Backward compatible: all existing positional usage continues to work
- Enhanced error handling for output file write operations

**Example Usage**:
```bash
# Traditional positional still works
diff-utility file1.txt file2.txt

# New named arguments
diff-utility -1 file1.txt -2 file2.txt -o output.txt

# Override: compares file1 with file3 (not file2)
diff-utility file1.txt file2.txt -file2 file3.txt
```
